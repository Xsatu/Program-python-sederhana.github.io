Menghitung Volume Balok

Hai,kali ini saya akan membuat program python sederhana yaitu membuat program menghitung volume balok. Langsung saja ke cara nya.
1.Buatlah folder dengan nama bebas
2.Buka kode editor kalian
3.Masuk kedalam folder yang sudah kalian buat
4.Buatlah file dengan nama bebas dan beri .py pada akhir nama file

Setelah itu ketikan print("Menghitung Volume balok") lalu ketikan p = int(input("Panjang :")). Karena rumus menghitung volume balok adalah v= p*l*t maka ketikan l = int(input("Lebar :")) dan t = int(input"Tinggi :")

Setelah itu kita buat rumus nya agar bisa di proses dengan mengetikan v = p*l*t

Untuk melihat hasilnya kalian harus mengetikan print("Volume Balok = %0i " "cm3" %v)

Untuk memunculkan output nya, bisa kalian ikuti langkah di bawah ini.
1.Buka command prompt
2.cari dimana kalian meletakan folder nya lalu ketikan cd nama letak folder
3.Jika sudah masukan nama folder
4.Setelah itu masukan nama file.py
5.Masukan angka-angka yang ingin di kerjakan
